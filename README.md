## Short Description

`aaaa_game` is a playful 2D mobile game made in Unity that keeps gameplay simple, fast, and surprisingly addictive. It blends quick reflex challenges with colorful visuals and easy tap controls, making it perfect for short sessions on the go.

## Getting Started

### Prerequisites

- Unity 2021.3 LTS (or a compatible Unity 2021.x version)
- Windows or macOS for Unity editor use
- Mobile device build target: Android / iOS

### Clone the repo

```bash
git clone https://github.com/your-username/aaaa_game.git
cd aaaa_game
```

### Open in Unity

1. Launch Unity Hub.
2. Click `Add` and select the repository folder.
3. Open the project in Unity 2021.3 LTS.
4. Set your build target to `Android` or `iOS` in `File > Build Settings`.

## Controls

### Touch

- Tap: primary action / jump / select
- Swipe: move, dodge, or interact depending on level

### Gamepad / Keyboard (Editor Testing)

- `Space` / `Enter`: primary action
- `Arrow keys` / `WASD`: move or navigate menus
- `Esc`: pause / back

## Project Structure

- `Assets/` - main Unity project assets and scenes
- `Assets/Scripts/` - gameplay scripts and controllers
- `Assets/Scenes/` - Unity scene files for levels and menus
- `Assets/Prefabs/` - reusable game objects and UI prefabs
- `Assets/Screenshots/` - placeholder screenshots for the README
- `ProjectSettings/` - Unity project configuration files
- `Packages/` - Unity package manifest and package lock
